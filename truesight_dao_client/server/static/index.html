Auth

Every DAO action is authenticated by an RSA-2048 digital signature with SHA-256 hashing (RSASSA-PKCS1-v1_5). No API keys, no OAuth tokens — the signature is the authentication. Generate a keypair, register it with your email, and sign every payload you submit.
